Project

Insider Threat Detection Project

MSc Dissertation Project

This project focused on detecting potential insider threats from behavioural log data using machine learning and sequence-based modelling.

Overview

I developed an MSc dissertation project using the CERT r4.2 insider threat dataset to analyze user behaviour and identify suspicious activity patterns. The goal was to improve insider threat detection while supporting clearer, more interpretable analysis for security review.

Tools Used

  • Python
  • Jupyter Notebook
  • Pandas and NumPy
  • scikit-learn
  • TensorFlow / Keras
  • Streamlit
  • SHAP

What I Did

  • Prepared and analyzed behavioural log data from the CERT r4.2 dataset.
  • Built and evaluated a hybrid BiLSTM + CNN + attention model for insider threat detection.
  • Compared its performance against a logistic regression baseline.
  • Evaluated results using ROC-AUC, precision, recall, and F1-score.
  • Created a Streamlit dashboard with SHAP-based explanations to support analyst-friendly review.

Results

The hybrid model improved performance over the baseline and helped reduce false positives while keeping detection results interpretable through dashboard-based review and SHAP explanations.
